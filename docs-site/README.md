# Corely Docs Site (Mintlify)

Public, user-facing documentation for Corely lives in this directory.

## Prerequisites

- Node.js 20+
- pnpm (for running root scripts)

## Run locally

From this directory:

```bash
npx mint dev
```

From monorepo root:

```bash
pnpm docs:dev
```

Mintlify must be started from the directory that contains `docs.json` (`docs-site/`).
